# neo4j-graph-database-python-integration
This project demonstrates practical integration between a Neo4j graph database and Python. 
The work focuses on:
- Creating relationships between existing nodes
- Executing Cypher queries from Python
- Retrieving and processing graph data
- Implementing and deploying user-defined functions
- Applying regex for structured data extraction
This project highlights backend data interaction and graph-based data modeling.
---
## Technologies Used
- Python 3
- Neo4j Graph Database
- Cypher Query Language
- py2neo (Python-Neo4j integration)
